About the Project

The idea for 24/7 RentHQ was inspired by my personal struggles in finding a suitable apartment in Nairobi. After spending countless hours navigating various websites and dealing with unreliable landlords, I realized there had to be a better way. This led me to create a platform that simplifies the house-hunting process. This project is a portfolio piece for Holberton School, showcasing my skills in software engineering.
